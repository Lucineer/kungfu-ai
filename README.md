# KungFu AI — Skill Injection for Repo-Native Agents

> *I know kung-fu. Load what you need.*

## What It Is

Skill injection for repo-native agents. A fresh vessel is like Neo before the upload — capable but unskilled. KungFu provides the skill files. Fork a template, load its skills, start at Level 1 instead of Level 0.

## Skill Levels

- **Level 0**: Fresh vessel (blank canvas)
- **Level 1**: Template loaded (domain vocabulary, common patterns)
- **Level 2**: Practiced (user-specific context accumulated)
- **Level 3**: Expert (novel connections, predictive capabilities)
- **Level 4**: Master (genuinely irreplaceable)

## The Math

Skill level grows logarithmically with interactions. But VALUE grows exponentially. The inflection point: ~1000 interactions. Before that, a vessel is a novelty. After, it's a necessity.
